Алгоритм действий для запуска программы через файл:
1. Скачайте архив с программой
2. Распакуйте папку из архива
3. Откройте распаковавшуюся папку
4. Запустите PhotoEdit.exe
Программа работает только на OC Windows, не пытайтесь запустить её на других OC
Не удаляйте любые файлы из папки, это приведёт к ошибкам
Выполняйте действия только в той последовательности, которая указала программа. 
Минимальные системные требования:
- Процессор с частотой 1 ГГц или быстрее, или система на кристалле SoC
- 1 ГБ для 32-разрядных систем или 2 ГБ для 64-разрядных систем
- Место на жёстком диске 16 ГБ для 32-разрядных систем или 32 ГБ для 64-разрядных систем
- Видеоадаптёр должен быть DirectX9 или более поздней версии с драйвером WDDM 1.0
- Дисплей не менее 800 х 600

Алгоритм действий для запуска программы напрямую через язык программирования Python:
1. Установите или откройте python версии 3.9 или 3.10
2. Откройте терминал в проекте
3. Введите в терминал терминал команду: pip install -r requirements.txt
4. Запустите скрипт pril.py
Все примечания распространяются и на запуск программы напрямую
