Название проекта - PhotoEdit
Основная задумка - мини редактор фотографий. В проекте реализованы простейшие функции по работе с фотографиями. 
В файле scripts.py написаны функции по обработке изображений, а в файле pril.py лежит сама программа. В папке UI лежат файлы с интерфейсом программы. В папке fonts лежат встроенные шрифты.
Проект залит на github. В github 2 ветки - main и info. В main ветке лежит сам проект. В info ветке лежат изображения для примера, пояснительная записка и собственно этот readme файл
В пояснительной записке написана инструкция как запустить программу через exe файл или напрямую
Реализован следующий функционал: 
1. Переворот фотографии
2. Добавление различных фильтров
3. Нахождение и выделение границ
4. Наложение водяного знака
5. Скругление фотографии
6. Добавление текста на фотографию(с возможностью использования своего шрифта)
7. Просмотр фотографий
8. Меню с информацией о программе, как ей пользоваться и техническая информаця
9. Извлечение каналов RGB или изменение их последовательности
